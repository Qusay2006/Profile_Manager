# Profile Manager

**********************************************************
> Note: This app uses a free mock REST API (dummyjson.com).
> Add/Edit return success responses from the API but changes are not persisted.
> this is expected behavior with mock APIs.
> Delete operations update the UI instantly.
**********************************************************

A Flutter mobile app for managing user profiles with full CRUD operations.

## Features
- View list of users fetched from REST API
- Add new users
- Edit existing users
- Delete users
- Real-time UI updates after every operation

## Tech Stack
- Flutter
- Dart
- REST API (dummyjson.com)

## Getting Started
1. Clone the repo
2. Run flutter pub get
3. Run flutter run
